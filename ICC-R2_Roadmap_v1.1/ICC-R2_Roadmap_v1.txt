ABSTRACT
--------
This document defines the implementation protocol for Information-Copying 
Cosmology (ICC-R2), a framework where spacetime and cosmological dynamics 
emerge from stochastic copying of a simplicial complex with defects.

Key results (Q2 2026):
• Defect parameter Δ* = 0.105 ± 0.005 derived from reaction-diffusion 
  simulations on a 16³ lattice (Track 1)
• Geometric factor α = 3.00 calibrated to H(z) suppression (Track 4)
• Successful integration of ICC background component into CLASS v3.3.4
• Calibrated linear response: H₀(μ) = H₀^ΛCDM × (1 + 0.5 μ) for μ ∈ [0, 0.05]
• Bridge script: Δ* → μ = α·Δ* → H(z) predictions

REPRODUCIBILITY
---------------
• All scripts accept --seed for deterministic runs
• Results logged in structured JSON format
• CLASS integration verified with background_verbose=1
• Python dependencies: numpy, scipy, matplotlib, classy (optional)

QUICK START
-----------

python scan_mu_icc.py
